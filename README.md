# myNowPlaying

自定义歌曲组件，有多种风格

- 歌曲封面
- 自定义内容
- 歌曲名称
- 歌手名称
- 歌词信息
- 播放进度

原项目链接：[now-playing-service](https://github.com/Widdit/now-playing-service.git)

## 使用

### 方法一

将源代码拉到本地，然后运行

1. `yarn install`
2. `yarn run dev`

### 方法二

直接使用在线链接 <https://lililiaa.github.io/myNowPlaying/>

### 方法三

下载doc.zip后直接打开index.html
